# Jogo com base em desviar do inimigo.

Jogo desenvolvido com HTML, CSS e JavaScript onde o objetivo é desviar dos obstáculos.

##  Funcionalidades
- Movimentação com teclado (A/D ou setas)
- Sistema de pontuação
- Recorde salvo no navegador
- Dificuldade progressiva
- Reinício da partida

##  Tecnologias
- HTML5
- CSS3
- JavaScript

## Objetivo
Projeto criado para praticar lógica de programação, manipulação do DOM e interações em jogos web.

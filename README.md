# Desvie dos Inimigos

Jogo interativo desenvolvido com HTML, CSS e JavaScript, onde o jogador deve desviar de obstáculos que caem continuamente, sobrevivendo o máximo de tempo possível.

##  Demonstração
Abra o arquivo `index.html` no navegador para jogar.

## Funcionalidades
- Movimentação com teclado (A/D ou setas)
- Sistema de pontuação em tempo real
- Recorde salvo no navegador (LocalStorage)
- Dificuldade progressiva (aumento de velocidade e spawn)
- Sistema de colisão entre elementos
- Reinício da partida

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## Conceitos aplicados
- Manipulação do DOM
- Lógica de jogos (loop, colisão, spawn)
- Controle de eventos (teclado)
- Armazenamento local (LocalStorage)
- Animações com `requestAnimationFrame`

##  Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, especialmente lógica de programação aplicada a jogos e interatividade no front-end.
